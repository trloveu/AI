## 项目介绍

本项目为作者在该专业相关课程内容的授课、学习过程中的总结，包括人工智能专业的所有课程教案、笔记、代码，并包括日常中网上看到的好的实例（如有侵权请告知删除）。课程内容和专业架构可能会随时调整。

1. **基础课程**
    - 1.1 零基础入门学习python
